Linux 學習筆記(一)

最近剛開始通過慕課網 (https://coding.imooc.com/class/154.html) 學習 Linux，光安裝就花了不少心思，爬了不少文章。這邊先將我第一天的學習筆記整理在這。一直到上完課應該是能出個三四篇學習筆記吧!

Linux 簡介

Linux 是什麼

  • Linux 是一種自由和開放原始碼的類 UNIX 作業系統
  • Linux 嚴格來講是單指作業系統的核心，因作業系統中包含了許多用戶圖形和其他實用工具
  • 常見的 Linux 發行版包刮 Red Hat Enterprise Linux, CentOS 或 Ubuntu 等

Linux 能做什麼

  • 企業服務器
  • 嵌入式開發

與 Windows 有甚麼不同

  • 沒有 exe 安裝程序
  • 大小寫是要區分的
  • 一切皆文件
  • 文件後綴名不是那麼重要，只是為了好識別而已

CentOS 的安裝

虛擬機是什麼

  • 科研解釋: 虛擬機 (Virtual Machine) 是指通過軟件模擬的具有完整硬件系統功能的、運行在一個完全隔離環境中的完整計算機系統
  • 白話解釋: 通過軟件技術，可以在自己的電腦上安裝另一個操作系統，而不影響原生系統

虛擬機軟件

  • VirtualBox
  • vmware

安裝虛擬機

  • 下載 VirtualBox (V5.2.16): https://www.virtualbox.org/wiki/Downloads

SSH

SSH 是甚麼

  • SSH: Secure Shell 安全外殼協議
  • 建立在應用層基礎上的安全協議
  • 可靠，專為遠程登錄會話和其他網絡服務提供安全性的協議
  • 有效防止遠程管理過程中的信息洩露問題
  • SSH 客戶端適用於多種平台
  • SSH 服務端幾乎支持所有 UNIX 平台

服務器安裝 SSH 服務

  • 安裝 SSH
    • yum install openssh-server
  • 啟動 SSH
    • service sshd start
  • 設置開機運行
    • chkconfig sshd on

客戶端安裝 SSH 工具

  • SSH 是典型的客戶端和服務端的交互模式，客戶端廣泛的支持各個平台
  • Windows 有很多工具可以支持 SSH 連接功能，例如: XShell, Putty, secureCRT
  • Linux 平台需要安裝客戶端軟件
    • yum install openssh-clients

SSH config 講解

  • config 為了方便我們批量多個 ssh
  • config 存放在 ~/.ssh/config
  • config 配置語法

SSH config 語法關鍵字

Host 別名
HostName 主機名
Port 端口
User 用戶名
IdentityFile 密鑰文件的路徑

SSH 安全免密登錄: ssh key

  • ssh key 使用非對稱加密方式生成公鑰和私鑰
  • 私鑰存放在本地 ~/.ssh 目錄
  • 公鑰可以對外公開，放在服務器的 ~/.ssh/authorized_keys

Linux 平台生成 ssh key

  • ssh-keygen -t rsa
  • ssh-keygen -t dsa

SSH 安全端口

  • 端口安全指的是盡量避免服務器的遠程連接端口被不法份子知道，為此而改變默認服務端口號的操作

  • 如何改變 SSH 服務端口

    修改 /etc/ssh/sshd_config 配置
