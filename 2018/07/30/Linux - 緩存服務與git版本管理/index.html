Redis 安裝

  • 獲取源碼

    http://download.redis.io/releases/redis-4.0.10.tar.gz

  • 解壓

    tar xvfz redis-4.0.10.tar.gz

  • 安裝

    yum install gcc

    make MALLOC=libc

    sudo make install

  • 啟動

    cd src

    ls

  • | 命令 | 簡介 |
    | :————–: | :———————————————————: |
    | redis-server | Redis 服務端啟動程序 |
    | redis-cli | Redis 客戶端操作工具，也可以用telnet 根據其純文本協議來操作 |
    | redis-benchmark | Redis 性能測試工具 |
    | redis-check-aof | 數據修復工具 |
    | redis-check-dump | 檢查導出工具 |

    啟動: ./redis-server

    Redis 基本命令學習: http://www.runoob.com/redis/redis-tutorial.html

Redis 拓展知識

  • Redis 不僅僅支持簡單的 k/v 類型的數據，同時還提供 list, set, hash 等數據結構的存儲

  • Redis 支持數據的備份，即 master-slave 模式的數據備份

  • Redis 支持數據的持久化，可以將內存中的數據保持再磁盤中，重啟的時候可以再次加載進行使用

Git 版本管理工具

Git 安裝與生成 SSH KEY

  • 安裝

    sudo yum install git

  • 生成 SSH KEY

    ssh-keygen

  • git 學習網址: https://git-scm.com/book/zh-tw/v2

Git 常用命令

  • git config: 讓你能夠取得和設定組態參數
  • git init: 初始化儲存庫
  • git clone: 取得現有 git 倉儲的複本
  • git remote: 了解目前已加進來的遠端版本控制倉庫
  • git fetch: 同步遠端伺服器上的資料到本地
  • git commit: 提交版本
  • git rebase: 重新定義分支的版本庫狀態
  • git pull: 取回遠程主機某個分支的更新，再與本地的指定分支合併
  • git push: 將本地分支的更新，推送到遠程主機

git 命令自動補全

  • 下載源碼，使用下載源碼中的 git-completion.bash 自動補全命令的文件

    git clone git@github.com:git/git.git

  • 複製 git-completion.bash 文件

    cp contrib/completion/git-completion.bash /etc/bash_completion.d/

  • 加載 bash 腳本

    source /etc/bash_completion.d/git-completion.bash

  • 自動加載腳本，編輯 ~/.bash_profile

    # Git bash autoload
if [-f /etc/bash_completion.d/git-completion.bash]; then source /etc/bash_completion.d/git-completion.bash
fi
