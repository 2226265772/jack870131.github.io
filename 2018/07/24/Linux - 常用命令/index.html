Linux - 常用命令

Linux 常用命令

軟件操作命令

  • 軟件包管理器: yum
  • 安裝軟件: yum install xxx
  • 卸載軟件: yum remove xxx
  • 搜索軟件: yum search xxx
  • 清理緩存: yum clean packages
  • 列出已安裝: yum list
  • 軟件包信息: yum info xxx

服務器硬件資源信息

  • 內存: free -m
  • 硬盤: df -h
  • 負載: w/top
  • cpu 個數和核數: cat /proc/cpuinfo (cat 命令用於連接文件並打印號標準輸出設備上)
  • fdisk: 觀察硬碟實體使用情形

Linux文件目錄結構

  • 根目錄 /
  • 家目錄 /home
  • 臨時目錄 /tmp
  • 配置目錄 /etc
  • 用戶程序目錄 /usr

文件操作基本命令

命令 解釋
ls 查看目錄下的文件
touch 新建文件
mkdir 新建文件夾
cd 進入目錄
rm 刪除文件和目錄
cp 複製
mv 移動
pwd 顯示路徑

Linux 文本編輯器: Vim

  • 在服務器上我們是沒有類 Windows 的操作介面的，所有 GUI 介面都無法使用
  • Vim 是最出名的 Linux 文本編輯器，可以對文件修改操作
  • Vim 安裝: yum install vim

文件權限 4-2-1

權限 數字

命令 解釋
tail 從文件尾部開始讀
head 從文件頭部讀
cat 讀取整個文件
more 分頁讀取
less 可控分頁
grep 搜索關鍵字
find 查找文件
wc 統計個數

  • tar 命令

防火牆設置

  • 作用: 保護服務器安全
  • 設置防活牆規則
    • 開放80, 22端口
  • 關閉防火牆
命令 指令
yum install firewalld 安裝
service firewalld start 啟動
service firewalld status 檢查狀態
service firewalld stop/disable 關閉或禁用
