Thinking in Java - 學習筆記(二)

自動遞增和遞減

  • 自動遞增: ++

    自動遞減: - -

  • 兩操作符各有兩種使用方式，稱之為”前綴式” & “後綴式”

    前綴遞增: ++x

    後綴遞增: x++

    前綴遞減: - - x

    後綴遞減: x - -

  • 前綴遞增和前綴遞減: 先執行運算，再生成值

    後綴遞增和後綴遞減: 先生成值，再執行運算

  • 事例:

    pubilc class AutoInc {
    public static void main(String[] args) {
        int i = 1;
        print("i : " + i);
        print("++i : " + ++i);
        print("i++ : " + i++);
        print("i : " + i);
        print("--i : " + --i);
        print("i-- : " + i--);
        print("i : " + i);
    }
}
/* Output:
i : 1
++i : 2
i++ : 2
i : 3
--i : 2
i-- : 2
i : 1
*/

對象的等價性

  • Example:

    public class Equivalence {
    public static void main(String[] args) {
        Integer num1 = new Integer(100);
        Integer num2 = new Integer(100);
        System.out.println(n1 == n2);
        System.out.println(n1 != n2);
    }
}
/* Output:
false
true
*/

  • 引起此現象的原因為儘管對象的內容相同，對象的引用卻不同，== & != 比較的為對象的引用

  • 如果想比較兩對象內容是否相同，可使用 equals(), 但這個方法不適用於”基本類型” (byte, short, int, long 等等)，基本類型使用 == & != 即可

  • 但凡事都有例外，當創建自己的類時，會發生以下狀況:

    class Number {
    int i;
}

public class EqualsMethod {
    public static void main(String[] args) {
        Number n1 = new Number();
        Number n2 = new Number();
        n1.i = n2.i = 40;
        System.out.println(n1.equals(n2));
    }
}
/* Output:
false
*/

    發生此狀況的原因為當要測試的類為自定義類時，要依照自定義類所提供的 equals 來決定如何比較。如果自定義類沒有覆蓋 Object 的 equals 類的話，則 equals() 的默認行為依舊是比較引用，與 “==” & “!=” 的比較方式相同

移位操作符

  • 移位操作符的運算對象是二進制的”位”，只可用來處理整數類型(基本類型的一種)

  • 左移位操作符( << )按照操作符右側指定的位數將操作符左邊的操作數向左移動(在低位補0)

    “有符號”右移位操作符( >> )按照操作符右側指定的位數將操作符左邊的操作數向右移動(符號為正，在高位插入0, 符號為負，在高位插入1)

    “無符號”右移位操作符( >>> )無論正負，都在高位插入0

  • “移位”可與”等號”( <<= 或 >>= 或 >>>=) 組合使用。操作符左邊的值會移動由右邊的值指定的數，再將得到的結果給左邊的變量

package pers.jack.other;

public class BitwiseOperator {
    public static void main(String[] args) {
        int x = 10;
        System.out.println(Integer.toBinaryString(x));
        System.out.println(x >> 1);
        System.out.println(Integer.toBinaryString(x >> 1));
        System.out.println(x / 2);
        System.out.println();
        System.out.println(Integer.toBinaryString(x));
        System.out.println(x << 1);
        System.out.println(Integer.toBinaryString(x << 1));
        System.out.println(x * 2);
    }
}
/*
1010
5
101
5

1010
20
10100
20
*/
