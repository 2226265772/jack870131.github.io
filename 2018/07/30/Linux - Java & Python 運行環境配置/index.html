Linux - Java & Python 運行環境配置

Java 運行環境配置

Java

  • sudo yum -y install java-1.8.0-openjdk*
  • java -version: 查看 java 版本

Tomcat

  • 官網: http://tomcat.apache.org/
  • Tomcat 9 下載地址: http://ftp.mirror.tw/pub/apache/tomcat/tomcat-9/v9.0.10/bin/apache-tomcat-9.0.10.tar.gz
  • sh startup.sh: 啟動 Tomcat

Maven

  • 官網: http://maven.apache.org/

  • Maven 下載地址: http://ftp.tc.edu.tw/pub/Apache/maven/maven-3/3.5.4/binaries/apache-maven-3.5.4-bin.tar.gz

  • mvn -v: 查看 maven 版本

  • mvn clean package: 對文件進行打包
  • maven 學習地址: https://www.w3cschool.cn/maven/

Python 運行環境配置

安裝 Python

  • 下載 Python 安裝包
    wget https://www.python.org/ftp/python/3.6.1/Python-3.6.1.tar.xz

    xz -d Python-3.6.1.tar.xz

    tar xvf Python-3.6.1.tar

  • 安裝配置

    cd Python-3.6.1

    ./configure (報錯則輸入 yum install make gcc gcc-c++)

  • 編譯安裝

    make all

    make install (報錯則輸入 make && make install)

  • 驗證安裝

    python -v

    python3 -v

  • 設置 python 3.x 為默認版本

    mv /usr/bin/python /usr/bin/python.bak

    ln -s /usr/local/bin/python3 /usr/bin/python

  • 配置 yum (python 升級後，會發生 yum 命令無法使用的情形)

    • vi /usr/bin/yum

      將第一行”#!/usr/bin/python” 改為 “#!/usr/bin/python2.7”

    • vi /usr/libexec/urlgrabber-ext-down

      同上

軟件包管理工具 pip

  • pip 安裝教程

    https://www.liquidweb.com/kb/how-to-install-pip-on-centos-7/

  • pip 換成豆瓣源

    mkdir ~/.pip

    vim ~/.pip/pip.conf

    輸入以下配置

    [global]
timeout=60
index-url=https://pypi.doubanio.com/simple

python 虛擬環境安裝

pip install virtualenv

基本命令

virtualenv jack

source jack/bin/activate

Flask
